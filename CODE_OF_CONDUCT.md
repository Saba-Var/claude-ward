# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1. The full text is published at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: everyone interacting in this project's issues, pull requests, and other spaces
is expected to be respectful and considerate. Harassment and exclusionary behavior are not
tolerated.

## Reporting

To report a concern, email **vartasashvili94@gmail.com**. Reports are handled
confidentially, and you can expect a response within a few days.
